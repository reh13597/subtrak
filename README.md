# SubTrak

SubTrak is a full-stack subscription tracking application designed to help users manage their digital footprint and monthly spending. It provides a centralized dashboard to track, analyze, and optimize recurring expenses.

## Technologies
- **Frontend:** Next.js (TypeScript), Tailwind CSS, ShadCN UI
- **Backend:** Node.js API Routes, MySQL
- **Authentication:** AWS Cognito + Amplify
- **Hosting:** AWS Amplify
- **Database:** AWS RDS (MySQL)
- **Communications:** EmailJS
