# SubTrak

SubTrak is a responsive full-stack subscription tracking application designed to help users manage their digital footprint and monthly spending. It provides a centralized dashboard to track, analyze, and optimize recurring expenses.

## Tech Stack

Front-End:
- Next.js
- TypeScript
- Tailwind CSS
- ShadCN UI

Back-End:
- AWS RDS (MySQL) + S3
- Next.js API Routes

Authentication:
- AWS Cognito + Lambda

Deployment:
- AWS Amplify

APIs:
- Google Gemini API
- EmailJS
